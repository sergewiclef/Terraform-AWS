Welcome to the AWS Personal Learning Project!

This is a simple tool for uploading files for your project. Briefly describe the project below and then choose the file you want to upload.

The project aims at create a platform to send mass emails from a CSV list to be uploaded by users

The project uses the following AWS services to accomplish the above

  • S3 Static Web Hosting as a landing page.
  • An API Gateway to receive email send requests
  • A Python Lambda function to format the emails.
  • AWS SES to send emails.

Please upload the csv file with the following attributes

  • Recipient Names
  • REcipient Email Address
